This module disable all inactive sessions since a given delay / Descrizione in italiano non (ancora) disponibile

This module was written to be able to kill(logout) all inactive sessions since a given delay. On each request the server checks if the session is yet valid regarding the expiration delay. If not a clean logout is operated. </p>
